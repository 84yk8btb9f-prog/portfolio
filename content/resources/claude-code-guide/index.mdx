## How to read this

Start with **01 and 02** if you've never opened `~/.claude/` before — they anchor the rest.

Skip straight to **06, 07, 08** if you already have things running and want to know what to customize first.

**16** is the payoff section — it shows how every piece from the earlier sections chains together into real work.

## A note on honesty

Some of this setup is overkill for most people. Some of it is essential.

Where I've kept something running because it earns its keep, I say so. Where I've stopped using something and left it installed anyway, I say that too. You can't read 200 skills by their descriptions and figure out which ten matter — you have to live with them.

That's what this guide is: the result of living with it.
