## What it is

A personal framework for developing and testing AI agents that need browser access. Handles the scaffolding that every agent project needs: structured reasoning loops, tool dispatch, state management, Playwright integration, and run logging.

Built to reduce the boilerplate of starting a new agent project from zero.

## Stack

Python, Anthropic SDK, Playwright.
